css란 Cascading Style Sheets를 의미한다. Cascading : 계단식, 폭포같은, 연속적인 Style : 멋을 내다 Sheets : (종이) 한 장 => 계단식으로 스타일을 정의하는 문서!

웹페이지의 스타일을 정의한다

HTML: 웹 페이지의 콘텐츠를 구성하는 언어 CSS: 웹 문서의 콘텐츠에 스타일을 추가하는 언어(색상, 크기, 위치 등) => HTML과 함께 사용되지 않는 CSS는 단순 텍스트일 뿐이며, 웹브라우저를 통해 결과물을 확인하기 위해서는 반드시 HTML 문서 작성이 선행되어야 함

CSS는 웹문서를 꾸며주기 위해 사용하는 언어이다. CSS코드는 CSS파일에 작성한다. 확장자는 .css이다 CSS코드는 HTML문서에 스타일을 추가(정의)하는 언어이다.
