1. HTML(HyperText Markup Language)이란 HyperText:하이퍼링크를 통해 어떤 문서에서 다른 문서로 접근할 수 있는 텍스트 Markup:표시 Language: 언어 =>하이퍼텍스트와 콘텐츠를 표시해주는 언어 =>웹 페이지의 콘텐츠를 정의하기 위해 사용되는 언어 2. HTML 문서의 파일 확장자 --> .html 또는 .htm 파일을 수정하고 싶다면 --> 텍스트편집기(메모장,Brackets,VSCode 등) 결과를 확인하고 싶다면 --> 웹브라우저(크롬,파이어폭스,사파리,인터넷익스플로러,엣지) 3. 개발자 도구 -웹사이트 개발용 도구 -css코드 확인, 모바일 모니터링,네트워크 상태 점검, 스크립트 명령어 확인 등 다양한 기능을 개발자에게 제공 4. 코드 에디터 프로그래머가 프로그램 소스 코드를 편집하기 위해 사용하는 소프트웨어 5.태그 HTML코드에서 정보(콘텐츠)를 정의하는 형식 *단일태그: 시작과 끝을 구분할 필요가 없는 태그 6. 문서의 첫 부분에서 문서 유형을 지정하는 단일 태그 html 문서 시작 웹 브라우저가 알아야 할 정보들을 담아주는 태그 문자 인코딩 및 문서 키워드 등에 대한 요약 정보를 기입하는 단일 태그 문서의 제목을 쓰는 곳 브라우저 화면에 표시할 내용을 작성하는 곳 -텍스트 태그 -이미지 태그 -각종 사용자 인터페이스(버튼,인풋,드롭다운 등) html 문서 종료 7.텍스트 태그
