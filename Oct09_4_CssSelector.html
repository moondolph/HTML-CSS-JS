선택자{ 속성명: 속성값; } 선택자: 어떤 요소에 스타일을 적용 할 것인지에 대한 정보 선택자의 종류 -기본 선택자 -그룹 선택자 -특성 선택자 -결합 선택자 -의사 클래스 -의사 요소

기본선택자

1. 전체 선택자 모든 요소를 선택한다 *{ color: blue; } 2. 태그 선택자 주어진 이름을 가진 요소를 선택한다 p{ color:blue; } 3. 클래스 선택자 주어진 클래스 속성값을 가진 요소를 선택한다 .text{ color:blue; } 4. 아이디 선택자 주어진 id 속성값을 가진 요소를 선택한다 id는 고유한 식별자 역할을 하는 전역 속성이다 #topic{ color:blue; }

그룹 선택자

다양한 유형의 요소를 한꺼번에 선택할 때 사용한다. h1, p, div{ color:blue; }

선택자 우선순위

아이디>클래스>태그
